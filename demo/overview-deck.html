Skill 01 before / after
04

Same prompt — a support bot grounded in help docs, asked a question the docs don't cover.

Before raw prompt → model
"answer the customer's question using our help docs"
  • Invents a refund policy that doesn't exist — no grounding
  • Docs + question merged → injection risk
  • Switch model later → no portability
  • No tests, no evidence — fails in production
After with Prompt Optimizer
// GPT variant Answer using ONLY the HELP_DOCS. ### HELP_DOCS {docs} ### QUESTION {question} If not in docs → escalate=true Output JSON: {answer, quote, escalate}
  • Grounded + quotes source; escalates, never guesses
  • Docs delimited — injection-safe
  • Ships a 4-case test set + Claude/DeepSeek variants
Verdict: from “trust me” to a provable, reusable lift — grounded JSON, nothing invented, portable across models.
